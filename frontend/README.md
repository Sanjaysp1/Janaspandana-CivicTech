🏛️ Janaspandana - AI-Powered CivicTech System

Janaspandana is an advanced, integrated Public Grievance Redressal System designed for the Government of Karnataka. This platform bridges the gap between citizens and nodal officers by utilizing Artificial Intelligence to automatically categorize issues, detect urgency, extract hazard metadata, and auto-route complaints via a live GPS heatmap system.

✨ Key Features

👤 Citizen Portal

Live Camera Anti-Spoofing: Enforces live photo capture to prevent fake reports and gallery spoofing.

AI Vision Extraction: Uses Google Gemini AI to instantly analyze photos, estimate pothole depth, detect bio-hazards, and extract text/context.

Native GPS Geo-Tagging: Interactive OpenStreetMap/Leaflet integration to pinpoint exact street coordinates.

Multilingual Voice Dictation: Voice-to-text support with automatic Kannada-to-English translation for officials.

Official PDF Affidavits: Instantly generates multi-page, print-ready, legally formatted PDF affidavits with official e-seals and auto-cropped evidence grids.

🛡️ Official Nodal Dashboard

AI Civic Intelligence: Real-time scoring and predictive machine learning models to forecast hazard surges.

Live Density Heatmap: Interactive severity mapping of active grievances across wards and districts.

Smart Auto-Assignment: Intelligent routing of complaints to specific departments (BBMP, BESCOM, BWSSB) based on AI context extraction.

Performance Matrix: SLA tracking and citizen-rating analytics for head workers and field engineers.

🛠️ Technology Stack

Frontend: React (Vite), Tailwind CSS, Lucide Icons, Leaflet.js (Native Maps)

Backend: Python, FastAPI, Uvicorn, Pandas

Artificial Intelligence: Google GenAI SDK (gemini-2.5-flash-preview)

Storage: JSON Document Engine (In-Memory/File fallback)

🚀 Installation & Setup

Prerequisites

Node.js (v18+)

Python 3.9+

A valid Google Gemini API Key

1. Backend Setup (FastAPI)

Open a terminal and navigate to the backend directory:

cd backend


Install the required Python dependencies:

pip install fastapi uvicorn pydantic pandas google-genai


Set your Gemini API key as an environment variable (or place it in a .env file):

# Windows
set GEMINI_API_KEY="your-api-key-here"

# Mac/Linux
export GEMINI_API_KEY="your-api-key-here"


Run the FastAPI server:

uvicorn main:app --reload --host 0.0.0.0 --port 8000


2. Frontend Setup (React)

Open a separate terminal and navigate to the frontend directory:

cd frontend


Install the node dependencies:

npm install


Start the React development server:

npm run dev


The application will be available at the local network link provided in the terminal (usually http://localhost:5173).

🧪 Demo Access

The system comes with a built-in one-click demo mode to test both sides of the platform.
